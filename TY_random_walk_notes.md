# TY Random Walk Tests

This file explains the Random Walk Test analysis for the TY 10-Year U.S. Treasury Note Futures market.

## Files

- `random_walk_TY.py`: Python code for running the tests.
- `random_walk_TY_summary.csv`: Summary results from the tests.
- `variance_ratio_TY.png`: Variance Ratio test plot.
- `push_response_TY.png`: Push-Response test plot.

## Data

The input file `TY-5minHLV.csv` is not uploaded because it is course-provided data.

To run the code, place `TY-5minHLV.csv` in the same folder as `random_walk_TY.py`.

## How to run
```bash
python random_walk_TY.py
```

For result interpretation, see `TY_random_walk_analysis.md`.
